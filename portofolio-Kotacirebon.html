Web ini terdapat basemap layers terdapat delapan basemap



Disini Ada beberapa konten web ada 3 masing-masing konten cricle, Rectangle dan polygon



Di samping itu, Juga terdapat segmen-segmen garis yang bersinggungan dengan lokasi area polygon

Dan Juga ada sembilan Custom Marker di custom marker tersebut terdapat informasi dan juga foto tempat tersebut


Demo : Klik disini

Fitur unggulan:

  • Melihat nama serta foto yang terdapat di loakasi tertentu
  • Bisa Mengetahui Seberapa Besar lokasi tersebut
  • Bisa Melihat berbagai Basemap

Bahan pembuatan

  • HTML sebagai kerangka
  • CSS sebagai pemanis dan layout
  • Javascript dan Jquery sebagai interaksi langsung
  • Leaflet Sebagai library sebagai membangun aplikasi
